&emsp;二人の〝そう〟の温度感が違いすぎてきつい。ローリーはともかく、<ruby>八意<rt>やごころ</rt></ruby>先生の目までが、何処か疑いの色を持っているように思えた。ボクが、チルノの体をバラ／＼にしているとでも言うのだろうか。自分で被害者をバラ／＼にしておいて医者に連れて行くとかサイコパスの極みみたいなことするわけないじゃないか。</br>

「使いっぱしりにしやがってと怒られましたよ」</br>
「ふうん？&emsp;まあ、それは今はいいわ。それよりも、彼女」</br>

&emsp;そういって、チルノに布団をかけてあげてと指示する<ruby>八意<rt>やごころ</rt></ruby>先生。暑いのがいけないのに、布団をかけるの？&emsp;そう思ってしまったが、布団自体は冷たければ冷たいものを保温するのだ、彼女の体温を保つ観点では布団は氷室と同意だ。例年夏にはあちー／＼といって分厚い布団にくるまっているのをよく見た。ボクがチルノにとっておきの冬用羽毛布団をかけてあげていると、<ruby>八意<rt>やごころ</rt></ruby>先生は彼女の状態の説明を始めた。</br>
